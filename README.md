# Введение в тестирование

Проект по основам работы QA-инженера по ручному тестированию.

## О проекте

Изучены фундаментальные аспекты тестирования ПО: от базовых понятий до анализа жизненного цикла разработки и философии тестирования. Рассмотрены различия между Quality Assurance (QA) и Quality Control (QC), применение стандартов ISTQB.

## Что выполнено

- Изучен ресурс ISTQB, анализ разделов Certifications, SCR и Glossary
- Сравнение официального и альтернативных определений тестирования
- Сравнительный анализ QA и QC с примерами из реальных вакансий
- Изучены методологии: Waterfall, V-model, Incremental, Scrum, Kanban
- Разбор 4 идей Agile-манифеста
- Исследованы 7 принципов тестирования с ранжированием и дополнениями

## Технологии

- QA/QC теория
- Методологии SDLC
- Agile
- ISTQB стандарты
